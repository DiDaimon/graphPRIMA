## Установка
Требования:
    - [Python 3.8 или свежее](https://www.python.org/downloads/release/python-3810/)
    - [Git](https://git-scm.com/download/win)
```bash
mkdir d:\Python
cd /d d:\Python
git clone https://github.com/DiDaimon/graphPRIMA.git
cd graphPRIMA
install.bat
```
